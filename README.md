# FileIngestFront

## Development server

`ng serve` -> `http://localhost:4200/`

## PROD / DOCKER ENV

`ng serve --host 0.0.0.0 --port 5001 --prod --configuration=production`

## GCP ENV (RUN)

`ng serve --docker --prod --configuration=gcp`
